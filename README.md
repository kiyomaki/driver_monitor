# driver_monitor
My research at Fujii Laboratory, Chiba Institute of Technology

## 概要
運転中の不注意を防止するための目の開閉検知および警告システムの研究

## 動作環境
- OS: Windows
- 環境: Visual Studio Code(powershell)
- 使用言語/ライブラリ
  - python 3.12.4
  - mediapipe
  - cv2
  - numpy

## 機能
- `looking_away`
画面を注視していない際に、画面に「LOOKING AWAY!」と警告を表示する。
- `perclos`
眠気を感知した際に、画面に「ASLEEP!」と警告を表示する。
